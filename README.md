# Projet_SY43


Application Android développée dans le cadre du projet SY43.

## Description

Projet Android (Kotlin + Jetpack Compose) développé dans le cadre du cours SY43 à l'UTBM. Application mobile "couteau suisse" pour les étudiants du campus : EDT, sessions de révision collaboratives, réservation de salles, fils d'actualité par matière, annonces des associations, échanges de groupes/matières, maraudes étudiantes et menus du RU.

## Fonctionnalités

- Gestion multi-utilisateurs
- Intégration base de données
- Interface construite avec Jetpack Compose
- Animations via Jetpack Compose Animation API
- Utilisation du capteur : [ex. GPS / caméra / accéléromètre]
